# Topic Scoring Prompt

Use this prompt to score candidate topics for low-priced note products.

## Evaluation criteria
- Practical usefulness
- Suitability for a low-priced note
- Can be explained without face or voice content
- Potential for series expansion
- Relevance to the creator's knowledge area

## Output format
Return JSON with:
- `title`
- `score`
- `reasons`
- `concerns`
- `recommended`

This file is a placeholder. Refine the prompt once the scoring API is connected.
